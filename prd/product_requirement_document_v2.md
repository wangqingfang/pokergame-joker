# 无厘头德州扑克 PRD v2 — 长线留存与技能树系统

> 本文档为 v2 玩法升级设计，承接 `product_requirement_document.md` 的 MVP 版本。
> 目标：从一次性 demo 升级为具备留存能力的休闲单机游戏。
> 状态：设计评估稿，未进入排期。

---

## 0. 一句话目标

把"赢一局爽一下"的纯局内乐趣，升级为"持有虚拟资产 → 投资技能 build → 形成个人风格"的长线循环。

---

## 1. 可行性评估

### 1.1 强项
- **持久化经济**给每局意义（输了肉疼、赢了能投资），是德扑变体最缺的留存抓手。
- **24h 救济补给**是经典防止永久流失的手段，FIFA、皇室战争、原神都在用。
- **技能树流派分化**让"无厘头"主题落地——每个人玩出不同人格，社交话题感拉满。
- **概率技能 + 升级提成功率** = 持续金币消耗点，构成核心商业循环。

### 1.2 风险点
1. **难度曲线**：新手钱不够升级，老手钱花不完——需要软上限（见经济设计）。
2. **付费 vs 免费天花板**：高级技能若只能氪金抽卡解锁，单机/小圈子玩家会卡死。
   - 决策：金币能买到所有蓝/紫，仅 T3 橙卡需抽卡。
3. **AI 也得有技能**否则玩家碾压无趣；但 AI 用大招会让玩家觉得不公。
   - 决策：AI 只在"挑战关"才用高级技能，日常局保留原 6 基础技能。
4. **存档与作弊**：本地 UserDefaults 容易被改。MVP 阶段先本地存档，未来上线必须配账号系统。
5. **24h 锁机制**对单机休闲玩家偏严。
   - 决策：输光后给 3 次免费快速补给（每次冷却 4h），第 4 次起才是 24h。

### 1.3 不做项（MVP+留存阶段）
- ❌ PVP 实时对战（工程量翻 3 倍）
- ❌ 真实货币 IAP（合规与审核坑）
- ❌ 战令/赛季系统（先验证核心循环）

## 2. 经济系统

### 2.1 核心数值（初值，可调）

- 初始钱包：**2000 金币**（够打 4 局）
- 单局入场费：**500 金币** → 兑换为桌上 **1000 筹码**（2:1 比例制造紧张感）
- 输光阈值：钱包 < 500（不够再打一局）→ 触发"补给窗"
- 补给金额：**1500 金币**（够 3 局，不补到舒适区，逼用户升级技能）
- 补给冷却：**首次 4h、第二次 4h、第三次 4h，之后 24h**（软着陆）
- 单局赢得：桌上筹码按 **1:2 反向兑换** 回钱包（赢一局回本+小赚，连胜才能富）

### 2.2 金币去向（沉没点）
1. **入场费**（强制消耗）
2. **技能购买**（自愿）
3. **技能升级**（自愿）
4. **抽卡**（自愿）
5. **皮肤/牌桌外观**（远期）

只有 1 是强制，其他都是"自愿掉坑"，留出节奏。

### 2.3 状态机

```
钱包 ≥ 500 ──开局──▶ 局中 ──结算──▶ 钱包 ±X
                                      │
                                      ▼
                              钱包 < 500 ?
                                      │
                                      ▼
                              触发补给冷却
                              冷却 0 → 直接补 1500
                              冷却 > 0 → 显示倒计时锁定
```

## 3. 技能树系统

### 3.1 四大流派

- **暴力流（红 / Brute）**：物理压制 — 高伤害、高消耗、明牌威胁
  - 代表：全场翻倍盲注、强制跟注、撕牌
- **智谋流（蓝 / Mage）**：法术控场 — 信息操作、概率扭曲
  - 代表：偷看、换牌、改公共牌
- **守护流（绿 / Guardian）**：防御回血 — 减伤、续航、反弹
  - 代表：护盾、退筹码、免疫一次技能
- **诡谲流（紫 / Trickster）**：阴招暗算 — 慢性、隐蔽、后手
  - 代表：慢性毒（未来 N 手缓慢扣血）、伪装手牌、记号牌

### 3.2 树形结构（每流派 3 层 × 3 分支 = 9 节点）

```
        T1 基础(蓝)         T2 进阶(紫)          T3 终极(橙)
红派    重拳/震慑/挑衅  →  绝杀连招 ×3   →   红派旗舰技 ×1
蓝派    偷看/换牌/迷雾  →  操控大招 ×3   →   蓝派旗舰技 ×1
绿派    护盾/回血/反弹  →  团队加持 ×3   →   绿派旗舰技 ×1
紫派    慢毒/标记/隐身  →  组合阴招 ×3   →   紫派旗舰技 ×1
```

### 3.3 解锁条件

每层解锁需同时满足：
- **游戏时长门槛**（防止开局氪金一夜满级）
- **前置技能已购买**（树形依赖）
- **足够金币**

| 层级 | 颜色 | 解锁时长 | 单技能金币 | 升级 5 级累计成本 |
|---|---|---|---|---|
| T1 | 蓝 | 0h | 800 | 200/400/800/1600/3200 |
| T2 | 紫 | 累计 2h | 3000 | 800/1600/3200/6400/12800 |
| T3 | 橙 | 累计 8h + 成就 | 仅抽卡 | 碎片合成升级 |

### 3.4 概率技能 + 升级

所有技能默认成功率：
- T1：**60%**
- T2：**45%**
- T3：**30%**

每级升级 **+6% 成功率**，满 5 级 = +30%。
- T1 满级 = 90%
- T2 满级 = 75%
- T3 满级 = 60%

**永远不到 100%** — 保留无厘头的"翻车感"，这是品牌灵魂。

## 4. 抽卡系统（仅 T3 橙卡）

- **十连抽**：5000 金币 / 一张橙卡碎片包
- **保底**：50 抽必出 1 张完整橙卡
- **碎片合成**：同名橙卡碎片 ×10 = 完整技能；多余碎片可换"通用碎片"
- **限定卡池**：每月轮换主题（如"周星驰主题"出"食神"系列橙卡）

抽卡只用金币不用真钱——MVP+留存期定位"重肝轻氪"。

## 5. 技能管理与出战

### 5.1 新增页面（3 个）

#### A. 技能库（Skill Library）
- 网格展示已解锁技能，按流派分 Tab
- 点技能进详情：图标、流派、当前等级、成功率、金币升级按钮、描述
- 顶部显示钱包余额、累计游戏时长

#### B. 技能树（Skill Tree）
- 4 个流派 Tab，每 Tab 内 3 层 9 节点的树状图
- 灰色 = 未解锁、蓝边 = 可购买、金边 = 已拥有
- 点未解锁节点弹"需要 XX 时长 + XX 金币"提示

#### C. 出战配置（Loadout）
- **每局最多带 4 个技能**（关键限制——制造 build 选择压力）
- 上半屏：当前已选 4 格，可拖拽排序，对应游戏内 1/2/3/4 快捷键
- 下半屏：技能库列表，点击加入/移除
- 底部"开始游戏"按钮，进入牌局后只显示这 4 个技能

### 5.2 主菜单升级

```
[ 钱包: 2000 💰 | 时长: 1h23m ]

  ▶ 开始一局（入场 500）
  ⚙ 技能配置
  🌳 技能树
  🎴 抽卡
  📊 战绩
```

## 6. AI 平衡

- **日常局**：AI 沿用原 6 个基础技能（保持现有平衡）
- **挑战局（新模式）**：AI 也出技能，玩家通关给金币+碎片奖励
- **AI 角色绑定流派人设**：暴力 AI 只用红派、智谋 AI 只用蓝派 — 加深无厘头角色感

---

## 7. 落地优先级

| 阶段 | 内容 | 价值 |
|---|---|---|
| **P1** | 钱包 + 入场费 + 输光补给 + 简易技能购买（非树状）| 跑通经济闭环 |
| **P2** | 技能树 UI + 4 流派分化 + 升级系统 | 建立 build 多样性 |
| **P3** | 出战配置页（4 技能上限）+ AI 流派人设 | 完成核心循环 |
| **P4** | 抽卡 + T3 橙卡 + 限定池 | 长线留存 |
| **P5** | 成就系统 + 战绩 + 皮肤 | 商业化预热 |

P1 + P2 已足以把游戏从 demo 升级为可朋友圈传播的休闲产品。

---

## 8. 设计原则（防止跑偏）

1. **永远不到 100%**：技能再强也会翻车——品牌灵魂。
2. **金币能买到 80% 体验**：抽卡只锦上添花，不卡天花板。
3. **24h 补给只在真·输光时**：钱包 ≥ 入场费 不弹，避免恶心人。
4. **技能数量 ≠ 强度**：每局只能带 4 个，逼出 build 思考。
5. **AI 比玩家弱半档**：单机游戏要让人爽，但翻车段子要保留。

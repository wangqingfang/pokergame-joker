# PRD-P5：成就 + 战绩 + 皮肤

> 来源：`product_requirement_document_v2.md` § 七 落地优先级 — P5 阶段
> 前置：P4 已上线
> 阶段价值：商业化预热 + 自传播素材。

## 1. 成就
- 50+ 个成就，分类：胜利、build、技能、抽卡、特殊（如 "10 次翻车")。
- 完成奖励：金币 / 通用碎片 / 皮肤兑换券。

## 2. 战绩
- 每手回放：保留最近 50 手的关键事件（手牌、关键决策、技能命中/翻车）。
- 个人统计：胜率、最常用技能、最赚钱 build。
- 一键分享卡片（适配 iOS 分享面板）。

## 3. 皮肤
- 牌桌主题（默认绿、夜店红、赌场金）
- 牌背设计（默认 + 解锁款）
- 头像框（成就/抽卡解锁）

仅金币和成就解锁，**P5 不引入真实货币**。

## 4. 验收
- [ ] 成就触发与展示逻辑稳定。
- [ ] 战绩页可回看 + 分享。
- [ ] 至少 3 套牌桌、3 套牌背可切换。

## 5. 风险
- **成就解锁通知打扰**：默认仅在结算页弹，主菜单只显示徽章。
- **皮肤资源体积**：单套 < 2MB，避免包体膨胀。

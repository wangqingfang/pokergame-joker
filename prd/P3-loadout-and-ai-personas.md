# PRD-P3：出战配置（4 技能上限）+ AI 流派人设

> 来源：`product_requirement_document_v2.md` § 七 落地优先级 — P3 阶段
> 前置：P2 已上线（技能树 + 升级系统）
> 阶段价值：把"全装载"改为"4 技能 build 选择"，并让 AI 形成可识别的角色个性，完成核心循环。

---

## 1. 目标与非目标

### 目标
- 玩家每局最多带 **4 个技能** 进入牌桌（包含基础 6 技能 + 已购扩展技能合并的池子）。
- 提供"出战配置"页面：拖拽/点击切换 4 个槽位，可保存多套预设。
- 每个 AI 角色绑定一个流派人设，AI 释放的技能与人设一致。
- AI 平衡：日常局 AI 仍只用基础 6 技能子集（每个 AI 抽 2-3 个），但**视觉上区分流派**（颜色、台词、技能特效色）。

### 非目标
- 抽卡（P4）。
- 跨 AI 联动技能（P5+）。

---

## 2. 出战配置规则

- 4 个槽位，可放任意已拥有技能（基础或扩展）。
- 同一技能不可重复装载。
- 至少装 1 个才能开始游戏。
- 装载后进入牌局，技能栏只显示这 4 个 + 底部"未装载"灰色提示。
- 支持 3 套预设（暴力 build / 防守 build / 混合），快速切换。

---

## 3. AI 流派人设

| AI 角色 | 流派 | 人设关键词 | 技能池（从基础 6 中选） | 主色 |
|---|---|---|---|---|
| AI 0 | 暴力 | 鲁莽、爱 all-in | unlucky, chaos, swap | 红 |
| AI 1 | 智谋 | 阴险、爱偷看 | peek, swap, melon | 蓝 |
| AI 2 | 守护 | 稳健、爱跟注 | shield, melon, peek | 绿 |
| (后续) AI 3 | 诡谲 | 神秘、长线 | chaos, peek, unlucky | 紫 |

AI 释放技能时：
- 飘字带流派色
- 台词文案差异化（"狮子搏兔！"/"虚虚实实……"/"稳如老狗"/"嘿嘿嘿……"）
- 头像边框色 = 流派色

## 4. 数据模型扩展

```swift
struct Loadout: Codable, Identifiable {
    let id: UUID
    var name: String         // "暴力 build" / "防守 build" / 自定义
    var slots: [String?]     // 长度恒为 4，存技能 ID（基础或扩展）
}

struct LoadoutStore: Codable {
    var loadouts: [Loadout]  // 默认 3 个预设
    var activeId: UUID
}

extension Player {
    var school: School?      // 仅 AI 玩家有值
    var aiSkillPool: [SkillKind]  // AI 装载的 2-3 个基础技能
}
```

## 5. UI 设计

### 5.1 出战配置页（新）
- 顶部：3 个预设 Tab + "新建预设"按钮 + 钱包条。
- 上半屏：4 个槽位（大卡片），点击/拖拽编辑。
- 下半屏：技能池（基础 6 + 已购扩展，按流派分组），未装载的可点击加入空槽，已装载的灰色不可重复。
- 底部：
  - "保存"按钮（即时持久化）
  - "开始游戏（入场 500）"按钮 — 取代原主菜单的"开始一局"

### 5.2 主菜单更新
- "开始一局"按钮替换为"出战配置"，进入配置页后再开始游戏（一步走完 build → 入场）。
- 上一局的 build 自动选中。

### 5.3 牌桌页更新
- 玩家技能栏只显示 4 槽（含冷却 / 等级 / 成功率）。
- AI 头像加流派色边框；AI 释放技能时飘字按流派色。

## 6. 验收标准

- [ ] 3 个默认预设可用：暴力 / 防守 / 混合。
- [ ] 槽位可保存与切换，重启 App 后保留。
- [ ] 不装技能或重复装技能时按钮置灰提示。
- [ ] 游戏内技能栏严格只显示装载的 4 个。
- [ ] 4 个 AI 流派色在头像、飘字、台词上一致。

---

## 7. 风险与缓解

- **build 同质化**：4 槽限制要求每个流派至少有 2-3 个值得带的技能；若 P2 升级成本过高导致玩家长期只买得起 T1，需调低升级前几级的金币门槛。
- **AI 平衡**：AI 仍用基础 6 技能保证手感不变，仅视觉/台词分流派；若 P4 后让 AI 用扩展技能需重新做平衡。
- **新手引导**：首次进入"出战配置"页弹一次 1 步引导（"选 4 个技能开始第一局"）。
